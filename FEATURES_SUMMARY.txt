  📦 FITUR 1: EXPORT REPORT
  ─────────────────────────────────────────────────────────────────────

  ✅ Single Export
     - Export hasil scan individual
     - Format: JSON, CSV, TXT
     - Auto-download
     
  ✅ Bulk Export  
     - Export multiple scans sekaligus
     - Pilih scan dengan checkbox
     - Select all functionality
     - Format: JSON, CSV, TXT

  Cara Menggunakan:
  1. Buka halaman "Advanced Search"
  2. Pilih scan (centang checkbox)
  3. Klik tombol "📥 Export"
  4. Pilih format
  5. File otomatis terunduh!

═══════════════════════════════════════════════════════════════════════════

  🔎 FITUR 2: ADVANCED SEARCH & FILTER
  ─────────────────────────────────────────────────────────────────────

  ✅ Basic Search
     - Cari berdasarkan URL
     - Cari berdasarkan vulnerability description
     - Cari berdasarkan vulnerability type
     
  ✅ Advanced Filters
     - Severity: Critical, High, Medium, Low
     - Date Range: From - To
     - Sort By: Date, URL, Vulnerabilities, Risk Score
     - Sort Order: Ascending / Descending
     
  ✅ Kombinasi Filter
     - Semua filter dapat dikombinasikan
     - Pencarian sangat spesifik
     - Results real-time

  Cara Menggunakan:
  1. Buka halaman "Advanced Search" (menu sidebar)
  2. Ketik query di search bar (optional)
  3. Klik "Show Advanced" untuk filter tambahan
  4. Atur filter sesuai kebutuhan
  5. Klik "Apply Filters"
  6. Hasil muncul sesuai kriteria!

  🎯 QUICK GUIDE

  Export Single Scan:
  1. Klik "📥 Export" pada scan
  2. Pilih format
  3. Done!

  Export Multiple Scans:
  1. Centang scans yang diinginkan
  2. Klik "📥 Export (X)" di atas
  3. Pilih format
  4. Done!

  Search & Filter:
  1. Buka "Advanced Search" page
  2. Gunakan search bar atau filters
  3. Klik "Apply Filters"
  4. Results appear!
